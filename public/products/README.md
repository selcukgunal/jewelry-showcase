# Ürün Görselleri

Her ürün için kendi ID'siyle isimlendirilmiş bir klasör oluşturun:

```
public/
  products/
    501262/
      1.jpg   ← Ana görsel
      2.jpg   ← İkinci görsel (opsiyonel)
    501263/
      1.jpg
    501264/
      1.jpg
      2.jpg
```

`data/products.ts` dosyasındaki `images` dizisindeki yollar bu klasöre göre yazılmıştır.
